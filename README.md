# Data-Cleaning-Visualization

## Project Overview

This project focuses on cleaning, preprocessing, and visualizing sales data from the Superstore dataset using Python.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Features

* Data Cleaning
* Missing Value Handling
* Duplicate Removal
* Outlier Detection and Removal
* Data Visualization
* Exploratory Data Analysis

## Visualizations

* Bar Chart
* Histogram
* Pie Chart
* Scatter Plot
* Heatmap
* Boxplot

## Output

* Cleaned Dataset
* Visual Reports
* Business Insights

## Learning Outcomes

* Data Preprocessing
* Data Analysis
* Data Visualization
* Python Programming
